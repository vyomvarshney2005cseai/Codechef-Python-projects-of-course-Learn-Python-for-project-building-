# My Programming Portfolio: Course Projects

Welcome to my project repository! This repository contains a collection of projects built while progressing through the course, covering everything from core logic to advanced file handling. 

---

## 📂 Project Structure & Roadmap

### 📦 Beginner-Level Projects
*Focuses on core programming concepts, data structures, and conditional logic.*

*   **To-do list application:** A simple task manager to practice CRUD operations (Create, Read, Update, Delete).
*   **Quiz Game:** A trivia game utilizing user input, logic conditions, and basic scoring mechanics.
*   **Word counter:** A text processing tool designed to count words, characters, and spaces.
*   **Contact manager:** A terminal/app-based directory to store, search, and manage contact info.

### 📦 Intermediate-Level Projects
*Focuses on advanced logic and persistent storage using **File Handling**.*

*   **Inventory management system:** A tracking tool to manage product stock levels, saving state changes directly to disk.
*   **Student database manager:** A persistent tool to create, read, update, and delete student records.
*   **File encryption tool:** A security utility that encrypts and decrypts text files using basic cryptographic keys.
